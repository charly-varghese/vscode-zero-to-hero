# Module 06 – Final Project

## Stock Analysis SQL Database (VS Code–Centric)

### Overview

This project completes **Module 06 – SQL & Database Tools** of the  
**VS Code – Zero to Hero** program.

The objective is to design a **stock analysis database**, perform analytical SQL queries, and generate **BI-ready datasets** using VS Code and SQLite.

---

## Project Objectives

- Import stock price data into a SQL database
- Perform time-series and performance analysis
- Create reusable analytical SQL queries
- Export clean datasets for BI tools
- Maintain a GitHub-ready SQL portfolio asset
